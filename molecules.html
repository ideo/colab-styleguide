Hero Header

Code For: Hero Headers

Hero Header are page leads that are handled with the .hero class applied to some element (likely a div).

How to add a text block to a hero: Simply add the .hero-text class to a div within the .hero element in order to have a centered text box. Note that you muxt specify the the unit and .size*of* to give it a width. You will also need to add a background color to the .text-container element in order to get things looking spiffy.

How to fill a hero with an image: You can use the .img-hero class in a nested img tag to add an image that fills the hero. Or you can fill it with a color if you prefer...implementation is up to you. The example below uses an image to fill the space.

How to make the heroe span full document width: If elements and classes are used outside of a .container class, the elements and images will naturally stretch to fill the entire width of the document.

<div class="hero"> <div class="line"> <img class="img-hero"> <div class="hero-text color-bg--coral unit size1of2"> <h1>this is a headline about a possible project of the colab.</h1> </div> </div> </div>

What the above code does...

this is a headline about a possible project of the colab.

Tiles

Code For: Standard Tiles

Standard tiles are used as a type of snapshot/synposis for the page viewer. Viewers can peruse standard tiles to get a high level sense of things.

You create a standard tile by adding the .tile and .standard-tile classes to a div element.

Note about width: The .standard-tile will fill the width of its parent container.

<div class="line"> <div class="lastGroup"> <!-- First Tile --> <div class="unit size1of3"> <div class="tile standard-tile"> <img class="img-tile"> <div class="text-container"> <h3>Header A</h3> <p>Some descriptive text about a thing here.</p> <button>Act Now!</button> </div> </div> </div> <!-- Second Tile --> <div class="unit size1of3"> <div class="tile standard-tile"> <img class="img-tile"> <div class="text-container"> <h3>Header B</h3> <p>Some descriptive text about a thing here with a text only button.</p> <button class="text-only">Text-only button!</button> </div> </div> </div> <!-- Third Tile --> <div class="lastUnit size1of3"> <div class="tile standard-tile"> <img class="img-tile"> <div class="text-container"> <h3>Header C</h3> <p>Some more descriptive text about another thing.</p> <button>Click here</button> </div> </div> </div> </div> </div>

What the above code does...

Header A

Some descriptive text about a thing here.

Header B

Some descriptive text about a thing here with a text only button.

Header C

Some more descriptive text about another thing.

Code For: Project Tiles

Project tiles are used to highlight projects and the like for the viewer.

You create a prject tile by adding the .tile and .project-tile classes to a div element.

Note about width: The .project-tile will fill the width of its parent container.

<div class="lastGroup"> <div class="unit size1of2"> <div class="tile project-tile"> <img class="triangles"> <div class="text-container"> <h1>Project Header</h1> <p> A paragraph (from the Greek paragraphos, "to write beside" or "written beside") is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose. </p> </div> </div> </div> </div>

What the above code does...

Project Header

A paragraph (from the Greek paragraphos, "to write beside" or "written beside") is a self-contained unit of a discourse in writing dealing with a particular point or idea. A paragraph consists of one or more sentences. Though not required by the syntax of any language, paragraphs are usually an expected part of formal writing, used to organize longer prose.
