Code: A Note About the Page Container

The entire page is contained by an element with the .container class added. This element sets the max width of the page based on the container width.

NOTE: It is also important to note the use of the .lastUnit and .lastGroup classes in the grid. These are specifically used in the last elements of groups and classes to handle the padding of the last element.

<div class="container"> <div class="line"> <div class="lastUnit size1of1">...</div> </div> ...other page html... </div>

Code: A Non-nested Grid

This code snippet applies to any and all non-nested grids. Note how they are wrapped in a line tag, and that the first two divs have the unit class applied, while the last unit specifically has the lastUnit class applied. This handles padding for you.

<div class="line"> <div class="unit size1of3"></div> <div class="unit size1of3"></div> <div class="lastUnit size1of3"></div> </div>

Logos & Icons

Code: Large Logos

The logo classes are modular.

To handle large sizes, you can use the .logo--large or .logo--large__with-text classes in conjunction with the .logo class.

To handle coloring you can use the .logo--dark and .logo--light classes.

<i class="logo logo--large__with-text logo--dark"></i> <i class="logo logo--large logo--light"></i>

What the above code does...

Code: Small Logos

To handle small sizes, you only have access to the .logo--smallclass in conjunction with the .logo class.

To handle coloring you can use the .logo--gray and .logo--multi classes.

<i class="logo logo--small logo--gray"></i> <i class="logo logo--small logo--multi"></i>

Site Navigation

Code: Site Navigation

These are the styles for the primary site navigation. The site navigation links can either move with the page (see the code sample immediately below), or stay fixed to the top of the page (like the current page you're viewing).

Both code examples are shown below. Important note: The element containing the .main-nav element itself should sit outside of a .container class. This way the header stretches to the full document width.

<div class="main-nav"> <ul> <li><a class="main-nav-link">Home</a></li> <li><a class="main-nav-link">Examples</a></li> <li><a class="main-nav-link">Pages</a></li> </ul> </div>

What the above code does...

  • Home
  • Examples
  • Pages


Fixing the primary navigation to the top of the page: You can also fix the primary site navigation to the top of the page and force it to span the full document width by adding the .main-nav--fixed class to the element already containing the .main-nav class.

<div class="main-nav main-nav--fixed"> <ul> ... </ul> </div>
